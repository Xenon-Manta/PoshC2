##################
  PoshC2 Help
##################

The Help is split out into two main elements, "Pre-Implant Help" and "Post-Implant Help". This is basically depending on when you type help, will depend on the help that is provided. 

To obtain the help in either sections type ``Help`` or ``?``.

Pre-Implant Help
####################

Main Menu
###############
Use Implant by <id>, e.g. 1
Use Multiple Implants by <id>,<id>,<id>, e.g. 1,2,5
Use ALL Implants by ALL

Post-Implant Help
####################

Module Help
###############
Either look up the help online, using the provided documentation on readthedocs, or alternatively load the module and then use the inbuilt Powershell help for that module::
	
	LoadModule Invoke-Mimikatz.ps1
	Get-Help Invoke-imikatz
